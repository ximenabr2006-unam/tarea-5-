Distribución Gamma

Función Gamma

Antes de estudiar la distribución gamma, es fundamental comprender la función gamma, que es la base matemática de esta distribución y una generalización del factorial para los números reales.

Definición 1 La función gamma se define por medio de la siguiente integral impropia:

\[\Gamma(\alpha) = \int_{0}^{\infty} t^{\alpha -1} e^{-t} dt, \quad \alpha > 0\]

Esta integral impropia converge para todos los valores reales \(\alpha > 0\) y para todos los números complejos con parte real positiva.

Propiedades Fundamentales de la Función Gamma

1. Relación de Recurrencia

La propiedad más importante de la función gamma es:

\[\Gamma(\alpha +1) = \alpha \Gamma(\alpha)\]

2. Relación con el Factorial

Para números enteros positivos \(n\):

\[\Gamma(n+1) = (n)!\]

Descripción de la Variable Aleatoria y Definición

La distribución gamma es una familia de distribuciones continuas extremadamente versátil que surge naturalmente en múltiples contextos estadísticos y aplicados. Su importancia radica en su capacidad para modelar una amplia variedad de fenómenos que involucran tiempos de espera, sumas de variables aleatorias y procesos de llegada.

La distribución gamma es fundamental para modelar:

  • Tiempo hasta la k-ésima falla en sistemas con múltiples componentes
  • Tiempo de procesamiento de tareas que requieren múltiples etapas
  • Modelado de lluvias y precipitaciones acumuladas
  • Análisis de supervivencia con tasas de riesgo variables
  • Distribuciones a priori en estadística bayesiana
  • Suma de variables exponenciales independientes

Definición 2 Una variable aleatoria continua \(X\) sigue una distribución gamma si tiene función de densidad de probabilidad:

\[f(x) = \frac{(\lambda x)^{\alpha-1}}{\Gamma(\alpha)} \lambda e^{-\lambda x}, \quad x > 0\]

donde:

  • \(\alpha > 0\) es el parámetro de forma (shape parameter)
  • \(\lambda > 0\) es el parámetro de tasa (rate parameter)
  • \(\Gamma(\alpha)\) es la función gamma evaluada en \(\alpha\)

A una variable aleatoria con distribución gamma la denotamos como \(X \sim \text{gamma}(\alpha, \lambda)\)

Casos Especiales Importantes

La distribución gamma incluye varias distribuciones conocidas como casos especiales:

1. Distribución Exponencial

Cuando \(\alpha = 1\): \[\text{gamma}(1, \lambda) = \text{exp}(\lambda)\]

2. Distribución Chi-cuadrada

Cuando \(\alpha = \frac{\nu}{2}\) y \(\lambda = \frac{1}{2}\): \[\text{gamma}\left(\frac{\nu}{2}, \frac{1}{2}\right) = \chi^2(\nu)\]

3. Distribución Erlang

Cuando \(\alpha\) es un entero positivo \(k\): \[\text{gamma}(k, \lambda) = \text{Erlang}(k, \lambda)\]

Interpretación de los Parámetros

Parámetro de Forma (\(\alpha\))

  • \(\alpha < 1\): Distribución en forma de “J” invertida, altamente sesgada
  • \(\alpha = 1\): Distribución exponencial (decrecimiento monótono)
  • \(\alpha > 1\): Distribución unimodal con sesgo positivo
  • \(\alpha \gg 1\): Distribución aproximadamente simétrica (normal)

Parámetro de Tasa (\(\lambda\))

  • \(\lambda\) pequeño: Distribución más dispersa, valores más grandes probables
  • \(\lambda\) grande: Distribución más concentrada, valores más pequeños probables
  • Efecto de escala: \(\lambda\) actúa como un factor de escala inverso

Propiedades de la Función de Densidad de Probabilidad

La función de densidad de probabilidad de la distribución gamma puede escribirse como:

\[f(x) = \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\lambda x}, \quad x > 0\]

La cual, puede descomponerse en tres componentes principales:

  1. Factor de normalización: \(\frac{\lambda^\alpha}{\Gamma(\alpha)}\)
  2. Componente de potencia: \(x^{\alpha-1}\)
  3. Componente exponencial: \(e^{-\lambda x}\)

Análisis del Comportamiento en x = 0

El comportamiento de \(f(x)\) cuando \(x \to 0^+\) depende crucialmente del parámetro de forma \(\alpha\):

\[\lim_{x \to 0} f(x) = \lim_{x \to 0} \frac{(\lambda x)^{\alpha-1}}{\Gamma(\alpha)} \lambda e^{-\lambda x}\]

  • Si α > 1: \(\lim_{x \to 0} f(x) = 0\)
  • Si α = 1: \(\lim_{x \to 0} f(x) = \frac{\lambda}{\Gamma(1)} = \lambda\)
  • Si 0 < α < 1: \(\lim_{x \to 0} f(x) = +\infty\)

Moda de la Distribución

Para encontrar la moda, calculamos \(f'(x) = 0\):

\[f'(x) = \frac{\lambda^\alpha}{\Gamma(\alpha)} \left[(\alpha-1)x^{\alpha-2}e^{-\lambda x} - \lambda x^{\alpha-1}e^{-\lambda x}\right]\]

\[= \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-2} e^{-\lambda x} \left[(\alpha-1) - \lambda x\right]\]

Igualando a cero: \((\alpha-1) - \lambda x = 0\)

  • Si α > 1: Moda = \(\frac{\alpha-1}{\lambda}\)
  • Si α = 1: No hay moda (distribución exponencial)
  • Si α < 1: Moda = 0

Derivada de la PDF

\[f'(x) = \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-2} e^{-\lambda x} \left[(\alpha-1) - \lambda x\right]\]

Análisis del signo:

  • Para \(x < \frac{\alpha-1}{\lambda}\): \(f'(x) > 0\) (función creciente)
  • Para \(x > \frac{\alpha-1}{\lambda}\): \(f'(x) < 0\) (función decreciente)

Comportamiento Asintótico

Cuando \(x \to \infty\): \[f(x) \sim \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\lambda x} \to 0\]

El decaimiento es exponencial, más rápido que cualquier función polinomial.

Propiedades de la Función de Distribución (CDF)

Definición 3 La función de distribución acumulativa (CDF) de la distribución gamma se define como:

\[F(x) = P(X \leq x) = \int_{0}^{x} \frac{(\lambda t)^{\alpha-1}}{\Gamma(\alpha)} \lambda e^{-\lambda t} dt\]

Para \(x > 0\), y \(F(x) = 0\) para \(x \leq 0\).

Medidas Descriptivas de la Distribución Gamma

Media (Esperanza)

La media de una distribución gamma \(X \sim \text{gamma}(\alpha, \lambda)\) se calcula como:

\[E(X) = \int_{0}^{\infty} x \cdot f(x) dx\]

Sustituyendo la función de densidad:

\[E(X) = \int_{0}^{\infty} x \cdot \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\lambda x} dx\]

\[= \frac{\lambda^\alpha}{\Gamma(\alpha)} \int_{0}^{\infty} x^\alpha e^{-\lambda x} dx\]

Para resolver esta integral, utilizamos la sustitución \(u = \lambda x\), donde \(du = \lambda dx\) y \(x = u/\lambda\):

\[E(X) = \frac{\lambda^\alpha}{\Gamma(\alpha)} \int_{0}^{\infty} \left(\frac{u}{\lambda}\right)^\alpha e^{-u} \frac{du}{\lambda}\]

\[= \frac{\lambda^\alpha}{\Gamma(\alpha)} \cdot \frac{1}{\lambda^{\alpha+1}} \int_{0}^{\infty} u^\alpha e^{-u} du\]

\[= \frac{1}{\lambda \cdot \Gamma(\alpha)} \int_{0}^{\infty} u^\alpha e^{-u} du\]

Reconocemos que \(\int_{0}^{\infty} u^\alpha e^{-u} du = \Gamma(\alpha + 1)\), y usando la propiedad de recurrencia \(\Gamma(\alpha + 1) = \alpha \cdot \Gamma(\alpha)\):

\[E(X) = \frac{1}{\lambda \cdot \Gamma(\alpha)} \cdot \alpha \cdot \Gamma(\alpha) = \frac{\alpha}{\lambda}\]

es decir,

\[E(X) = \mu = \frac{\alpha}{\lambda}\]

Interpretación:

  • La media es directamente proporcional al parámetro de forma \(\alpha\)
  • La media es inversamente proporcional al parámetro de tasa \(\lambda\)
  • Si \(\alpha\) aumenta (manteniendo \(\lambda\) fijo), la distribución se desplaza hacia la derecha
  • Si \(\lambda\) aumenta (manteniendo \(\alpha\) fijo), la distribución se concentra hacia valores menores

Varianza

La varianza de la distribución gamma se calculará usando la fórmula:

\[\text{Var}(X) = E[X^2] - (E[X])^2\]

Cálculo de \(E(X^2)\)

Primero necesitamos calcular el segundo momento:

\[E(X^2) = \int_{0}^{\infty} x^2 \cdot \frac{\lambda^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\lambda x} dx\]

\[= \frac{\lambda^\alpha}{\Gamma(\alpha)} \int_{0}^{\infty} x^{\alpha+1} e^{-\lambda x} dx\]

Usando la misma sustitución \(u = \lambda x\):

\[E(X^2) = \frac{\lambda^\alpha}{\Gamma(\alpha)} \cdot \frac{1}{\lambda^{\alpha+2}} \int_{0}^{\infty} u^{\alpha+1} e^{-u} du\]

\[= \frac{1}{\lambda^2 \cdot \Gamma(\alpha)} \int_{0}^{\infty} u^{\alpha+1} e^{-u} du\]

\[= \frac{1}{\lambda^2 \cdot \Gamma(\alpha)} \cdot \Gamma(\alpha + 2)\]

Usando \(\Gamma(\alpha + 2) = (\alpha + 1) \cdot \alpha \cdot \Gamma(\alpha)\):

\[E(X^2) = \frac{(\alpha + 1) \cdot \alpha \cdot \Gamma(\alpha)}{\lambda^2 \cdot \Gamma(\alpha)} = \frac{\alpha(\alpha + 1)}{\lambda^2}\]

Ahora, calculamos la varianza: \[\text{Var}(X) = E(X^2) - E(X)^2 = \frac{\alpha(\alpha + 1)}{\lambda^2} - \left(\frac{\alpha}{\lambda}\right)^2\]

\[= \frac{\alpha(\alpha + 1)}{\lambda^2} - \frac{\alpha^2}{\lambda^2} = \frac{\alpha(\alpha + 1) - \alpha^2}{\lambda^2}\]

\[= \frac{\alpha^2 + \alpha - \alpha^2}{\lambda^2} = \frac{\alpha}{\lambda^2}\]

es decir, \[\text{Var}(X) = \sigma^2 = \frac{\alpha}{\lambda^2}\]

Interpretación:

  • La varianza es proporcional al parámetro de forma \(\alpha\)
  • La varianza es inversamente proporcional al cuadrado del parámetro de tasa \(\lambda\)
  • Mayor \(\alpha\) implica mayor dispersión (para \(\lambda\) fijo)
  • Mayor \(\lambda\) implica menor dispersión (concentración hacia valores menores)

Desviación Estándar

La desviación estándar es la raíz cuadrada positiva de la varianza:

\[\sigma = \sqrt{\text{Var}(X)} = \sqrt{\frac{\alpha}{\lambda^2}} = \frac{\sqrt{\alpha}}{\lambda}\]

Coeficiente de Variación

El coeficiente de variación (CV) es una medida de dispersión relativa:

\[CV = \frac{\sigma}{\mu} = \frac{\frac{\sqrt{\alpha}}{\lambda}}{\frac{\alpha}{\lambda}} = \frac{\sqrt{\alpha}}{\alpha} = \frac{1}{\sqrt{\alpha}}\]

Interpretación del CV:

  • El coeficiente de variación solo depende del parámetro de forma \(\alpha\)
  • Es independiente del parámetro de tasa \(\lambda\)
  • A mayor \(\alpha\), menor variabilidad relativa
  • Para \(\alpha = 1\) (exponencial): \(CV = 1\)
  • Para \(\alpha \to \infty\): \(CV \to 0\) (distribución muy concentrada)

simulacion_TCL_gamma <- function() {
  cat("======================================================================\n")

  cat("======================================================================\n")
  
  # Parámetros de la distribución gamma
  alpha <- 2.5
  lambda <- 1.8
  
  # Media y varianza teóricas
  media_teorica <- alpha / lambda
  varianza_teorica <- alpha / (lambda^2)
  
  cat("Parámetros de la distribución Gamma:\n")
  cat("  Forma (α):", alpha, "\n")
  cat("  Tasa (λ):", lambda, "\n")
  cat("  Media teórica (μ = α/λ):", round(media_teorica, 4), "\n")
  cat("  Varianza teórica (σ² = α/λ²):", round(varianza_teorica, 4), "\n\n")
  
  # Tamaños de muestra
  tamanos_muestra <- c(5, 10, 20, 30, 50, 100)
  num_muestras <- 1000
  
  # Listas para almacenar resultados
  medias_muestrales <- list()
  varianzas_medias <- numeric(length(tamanos_muestra))
  
  # Configurar espacio para gráficos
  par(mfrow = c(3, 2))
  
  for (i in seq_along(tamanos_muestra)) {
    n <- tamanos_muestra[i]
    medias <- numeric(num_muestras)
    
    cat("Procesando n =", n, "...\n")
    
    # Generar 1000 muestras de tamaño n y calcular sus medias
    for (j in 1:num_muestras) {
      muestra <- rgamma(n, shape = alpha, rate = lambda)
      medias[j] <- mean(muestra)
    }
    
    # Almacenar resultados
    medias_muestrales[[i]] <- medias
    varianzas_medias[i] <- var(medias)
    
    # Parámetros según el TCL
    media_medias_teorica <- media_teorica
    varianza_medias_teorica <- varianza_teorica / n
    sd_medias_teorica <- sqrt(varianza_medias_teorica)
    
    # Estadísticas muestrales
    media_medias_muestral <- mean(medias)
    varianza_medias_muestral <- var(medias)
    
    # Crear histograma
    hist(medias, breaks = 30, freq = FALSE, 
         col = "lightblue", border = "white",
         main = paste("Distribución de Medias - n =", n),
         xlab = "Media muestral", ylab = "Densidad",
         xlim = c(media_teorica - 3*sd_medias_teorica, 
                  media_teorica + 3*sd_medias_teorica))
    
    # Superponer curva normal teórica según TCL
    x <- seq(media_teorica - 3*sd_medias_teorica, 
             media_teorica + 3*sd_medias_teorica, 
             length.out = 1000)
    y <- dnorm(x, mean = media_medias_teorica, sd = sd_medias_teorica)
    lines(x, y, col = "red", lwd = 2)
    
    # Línea para la media teórica
    abline(v = media_medias_teorica, col = "darkgreen", lwd = 2, lty = 2)
    
    # Línea para la media muestral
    abline(v = media_medias_muestral, col = "blue", lwd = 2, lty = 3)
    
    # Calcular medidas de calidad de la aproximación
    sesgo <- moments::skewness(medias)
    curtosis <- moments::kurtosis(medias)
    
    # Agregar información al gráfico
    legend("topright", 
           legend = c(paste("Media muestral:", round(media_medias_muestral, 4)),
                      paste("Varianza muestral:", round(varianza_medias_muestral, 4)),
                      paste("Teórico: N(", round(media_medias_teorica, 2), ", ", 
                            round(varianza_medias_teorica, 4), ")", sep = ""),
                      paste("Sesgo:", round(sesgo, 3)),
                      paste("Curtosis:", round(curtosis, 3))),
           bg = "white", cex = 0.7)
    
    # Imprimir resultados en consola
    cat("  Media de medias muestrales:", round(media_medias_muestral, 4), "\n")
    cat("  Varianza de medias muestrales:", round(varianza_medias_muestral, 4), "\n")
    cat("  Varianza teórica (σ²/n):", round(varianza_medias_teorica, 4), "\n")
    cat("  Diferencia en varianzas:", round(varianza_medias_muestral - varianza_medias_teorica, 4), "\n")
    cat("  Error relativo:", round(abs(varianza_medias_muestral - varianza_medias_teorica) / varianza_medias_teorica * 100, 2), "%\n\n")
  }
  
  par(mfrow = c(1, 1))
  
  # Gráfico adicional: Convergencia de varianzas
  dev.new()
  plot(tamanos_muestra, varianzas_medias, type = "b", pch = 19, col = "blue", lwd = 2,
       main = "Convergencia de la Varianza de Medias Muestrales",
       xlab = "Tamaño de muestra (n)", ylab = "Varianza de medias muestrales",
       ylim = c(0, max(varianzas_medias) * 1.1),
       log = "x")

  # Pero para la varianza de medias: Var(Var(𝑋̄)) ≈ 2(σ²/n)²/(m-1) donde m es número de muestras
  error_var <- sqrt(2 * (varianza_teorica/n_teorico)^2 / (num_muestras - 1))

simulacion_suma_exponenciales <- function() {
  cat("\n\n======================================================================\n")
  cat("EJERCICIO 2: SUMA DE VARIABLES EXPONENCIALES ~ GAMMA\n")
  cat("======================================================================\n")
  
  # Parámetro de la distribución exponencial
  lambda <- 1.5
  
  # Valores de n a simular
  n_valores <- c(5, 10, 20)
  num_simulaciones <- 1000
  
  # Media y varianza de una exponencial
  media_exp <- 1/lambda
  varianza_exp <- 1/(lambda^2)
  
  cat("Parámetros de la distribución Exponencial:\n")
  cat("  Tasa (λ):", lambda, "\n")
  cat("  Media (1/λ):", round(media_exp, 4), "\n")
  cat("  Varianza (1/λ²):", round(varianza_exp, 4), "\n\n")
  
  cat("Propiedad a verificar:\n")
  cat("  Si X_i ~ Exp(λ) i.i.d., entonces ΣX_i ~ Gamma(n, λ)\n")
  cat("  Media teórica de Gamma(n, λ): n/λ =", n_valores[1]/lambda, "para n =", n_valores[1], "\n")
  cat("  Varianza teórica de Gamma(n, λ): n/λ² =", n_valores[1]/(lambda^2), "para n =", n_valores[1], "\n\n")
  
  # Configurar espacio para gráficos
  par(mfrow = c(3, 3))
  
  for (i in seq_along(n_valores)) {
    n <- n_valores[i]
    
    cat("Simulando n =", n, "variables exponenciales...\n")
    
    # Parámetros teóricos de la Gamma(n, λ)
    alpha_teorico <- n  # Forma
    beta_teorico <- lambda  # Tasa
    
    media_gamma_teorica <- alpha_teorico / beta_teorico
    varianza_gamma_teorica <- alpha_teorico / (beta_teorico^2)
    
    # Vectores para almacenar resultados
    sumas_directas <- numeric(num_simulaciones)  # Sumando exponenciales
    valores_gamma <- numeric(num_simulaciones)   # Generando directamente Gamma
    
    # Generar simulaciones
    for (j in 1:num_simulaciones) {
      # Método 1: Sumar n variables exponenciales
      exponenciales <- rexp(n, rate = lambda)
      sumas_directas[j] <- sum(exponenciales)
      
      # Método 2: Generar directamente de Gamma(n, λ)
      valores_gamma[j] <- rgamma(1, shape = n, rate = lambda)
    }
    
    # Gráfico 1: Histograma de sumas vs densidad Gamma teórica
    hist(sumas_directas, breaks = 30, freq = FALSE, 
         col = "lightblue", border = "white",
         main = paste("Suma de", n, "Exp(", lambda, ")", sep = ""),
         xlab = "Valor de la suma", ylab = "Densidad",
         xlim = c(0, max(sumas_directas) * 1.1))
    
    # Densidad teórica Gamma
    x <- seq(0, max(sumas_directas), length.out = 1000)
    y <- dgamma(x, shape = n, rate = lambda)
    lines(x, y, col = "red", lwd = 2)
    
    # Media muestral y teórica
    media_sumas <- mean(sumas_directas)
    abline(v = media_sumas, col = "blue", lwd = 2, lty = 2)
    abline(v = media_gamma_teorica, col = "darkgreen", lwd = 2, lty = 3)
    
    legend("topright", 
           legend = c("Densidad muestral", "Densidad Gamma teórica",
                      paste("Media muestral:", round(media_sumas, 3)),
                      paste("Media teórica:", round(media_gamma_teorica, 3))),
           col = c("lightblue", "red", "blue", "darkgreen"),
           lwd = c(NA, 2, 2, 2), lty = c(NA, 1, 2, 3),
           fill = c("lightblue", NA, NA, NA),
           border = c("white", NA, NA, NA),
           cex = 0.7)
    
    # Gráfico 2: Q-Q plot sumas vs Gamma teórica
    qqplot(qgamma(ppoints(num_simulaciones), shape = n, rate = lambda), 
           sumas_directas,
           main = paste("Q-Q Plot: Suma vs Gamma(", n, ",", lambda, ")", sep = ""),
           xlab = "Cuantiles teóricos Gamma", 
           ylab = "Cuantiles muestrales suma",
           pch = 19, cex = 0.5, col = "blue")
    abline(0, 1, col = "red", lwd = 2)
    
    # Gráfico 3: Comparación directa Gamma vs Suma
    hist(valores_gamma, breaks = 30, freq = FALSE,
         col = rgb(0, 0.5, 0, 0.5), border = "white",
         main = paste("Comparación: Gamma vs Suma (n=", n, ")", sep = ""),
         xlab = "Valor", ylab = "Densidad",
         xlim = range(c(sumas_directas, valores_gamma)))
    
    hist(sumas_directas, breaks = 30, freq = FALSE,
         col = rgb(0, 0, 1, 0.5), border = "white", add = TRUE)
    
    # Densidad teórica
    lines(x, y, col = "red", lwd = 3)
    
    legend("topright", 
           legend = c("Gamma directa", "Suma de exponenciales", "Densidad teórica"),
           fill = c(rgb(0, 0.5, 0, 0.5), rgb(0, 0, 1, 0.5), NA),
           border = c("white", "white", NA),
           col = c(NA, NA, "red"), lwd = c(NA, NA, 3),
           cex = 0.7)
    
    # Estadísticas comparativas
    cat("  Resultados para n =", n, ":\n")
    cat("    Media de sumas:", round(mean(sumas_directas), 4), "\n")
    cat("    Media teórica Gamma:", round(media_gamma_teorica, 4), "\n")
    cat("    Varianza de sumas:", round(var(sumas_directas), 4), "\n")
    cat("    Varianza teórica Gamma:", round(varianza_gamma_teorica, 4), "\n")
    
    # Prueba de Kolmogorov-Smirnov
    ks_test <- ks.test(sumas_directas, valores_gamma)
    cat("    Prueba KS p-value:", round(ks_test$p.value, 4), "\n")
    
    if (ks_test$p.value > 0.05) {
      cat("    Conclusión: No se rechaza que sean la misma distribución ✓\n")
    } else {
      cat("    Conclusión: Se rechaza que sean la misma distribución ✗\n")
    }
    cat("\n")
  }
  
  par(mfrow = c(1, 1))
  
  # Gráfico adicional: Evolución de la forma con n
  dev.new()
  
  # Preparar datos para múltiples n
  n_extra <- c(1, 2, 5, 10, 20, 30)
  x_range <- seq(0, 30, length.out = 1000)
  
  plot(x_range, dgamma(x_range, shape = n_extra[1], rate = lambda), 
       type = "l", col = 1, lwd = 2,
       main = "Evolución de la forma de Gamma(n, λ) con n",
       xlab = "x", ylab = "Densidad Gamma(n, λ)",
       ylim = c(0, 0.6))
  
  # Colores para diferentes n
  colores <- rainbow(length(n_extra))
  
  for (i in 2:length(n_extra)) {
    n_temp <- n_extra[i]
    lines(x_range, dgamma(x_range, shape = n_temp, rate = lambda), 
          col = colores[i], lwd = 2)
  }
  
  legend("topright", 
         legend = paste("n =", n_extra),
         col = colores, lwd = 2, cex = 0.8)
  
  # Teorema Central del Límite aplicado a la suma
  cat("\n======================================================================\n")
  cat("OBSERVACIÓN SOBRE EL TCL PARA SUMAS DE EXPONENCIALES\n")
  cat("======================================================================\n")
  
  cat("Para la suma de n variables exponenciales i.i.d.:\n")
  cat("  Y = ΣX_i ~ Gamma(n, λ)\n")
  cat("  Media: E[Y] = n/λ\n")
  cat("  Varianza: Var(Y) = n/λ²\n")
  cat("\nSegún el TCL, para n grande:\n")
  cat("  (Y - n/λ) / sqrt(n/λ²) ≈ N(0, 1)\n")
  cat("  O equivalentemente: Y ≈ N(n/λ, n/λ²)\n")
  
  # Demostración con n = 30
  n_grande <- 30
  sumas_grandes <- numeric(1000)
  
  for (j in 1:1000) {
    exponenciales <- rexp(n_grande, rate = lambda)
    sumas_grandes[j] <- sum(exponenciales)
  }
  
  # Estandarizar
  sumas_estandarizadas <- (sumas_grandes - n_grande/lambda) / sqrt(n_grande/(lambda^2))
  
  dev.new()
  par(mfrow = c(1, 2))
  
  # Histograma de sumas estandarizadas vs N(0,1)
  hist(sumas_estandarizadas, breaks = 30, freq = FALSE,
       col = "lightblue", border = "white",
       main = paste("Sumas Estandarizadas (n=", n_grande, ")", sep = ""),
       xlab = "Suma estandarizada", ylab = "Densidad",
       xlim = c(-4, 4))
  
  x_norm <- seq(-4, 4, length.out = 1000)
  y_norm <- dnorm(x_norm)
  lines(x_norm, y_norm, col = "red", lwd = 2)
  
  # Q-Q plot
  qqnorm(sumas_estandarizadas, 
         main = paste("Q-Q Plot Sumas Estandarizadas (n=", n_grande, ")", sep = ""),
         pch = 19, cex = 0.5, col = "blue")
  qqline(sumas_estandarizadas, col = "red", lwd = 2)
  
  # Prueba de normalidad
  shapiro_test <- shapiro.test(sumas_estandarizadas)
  mtext(paste("Shapiro-Wilk p =", round(shapiro_test$p.value, 4)),
        side = 3, line = 0.5, cex = 0.8)
  
  par(mfrow = c(1, 1))
  
  cat("\nPara n =", n_grande, ":\n")
  cat("  Sesgo de sumas estandarizadas:", round(moments::skewness(sumas_estandarizadas), 4), "\n")
  cat("  Curtosis de sumas estandarizadas:", round(moments::kurtosis(sumas_estandarizadas), 4), "\n")
  cat("  (Normal: sesgo=0, curtosis=3)\n")
  
  if (shapiro_test$p.value > 0.05) {
    cat("  Conclusión: La aproximación normal es adecuada para n =", n_grande, "✓\n")
  } else {
    cat("  Conclusión: La aproximación normal NO es adecuada para n =", n_grande, "✗\n")
  }
  
  cat("\n======================================================================\n")
  cat("RESUMEN DEL EJERCICIO 2\n")
  cat("======================================================================\n")
  
  cat("Se ha verificado la propiedad:\n")
  cat("  Si X_i ~ Exp(λ) i.i.d., entonces Σ_{i=1}^n X_i ~ Gamma(n, λ)\n\n")

  cat("2. Para n grande, la distribución Gamma se aproxima a una normal\n")

  cat("4. La propiedad se mantiene para diferentes valores de n\n")
  
  cat("\nAplicaciones prácticas:\n")
  cat("• Tiempo hasta n eventos en un proceso de Poisson\n")
  cat("• Suma de tiempos de vida de componentes exponenciales\n")
  cat("• Modelado de tiempos de espera acumulados\n")
